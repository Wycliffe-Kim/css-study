# CSS 구성 블록

이번 과에서는 다음과 같은 내용들을 배운다.

- 계단식(cascade) 및 상속(interitance)
- 선택자
  - type, class 및 id 선택자
  - 속성 선택자
  - pseudo-class 및 pseudo-element
  - 결합자(combinator)
- 박스 모델: CSS 레이아웃을 구성하기 위해서 필요한 것
- 배경 및 테두리
- 다양한 텍스트 방향 처리
- overflow 속성
- CSS 값 및 단위
- CSS에서의 항목 크기 조정
- 이미지, 미디어 등을 다루는 법
- 표 스타일링
- CSS 디버깅
- CSS 관리
